## Installation

1. Open **Pear Desktop**
2. Click **Options** (top left)
3. Go to **Visual Tweaks**
4. Open **Themes**
5. Select **Import Custom CSS**
6. Choose the downloaded `theme.css` file
7. Apply and restart if required
